# 📷 Photo Gallery

A responsive single-page photo gallery app built with vanilla HTML, CSS, and JavaScript.

## Features
- Drag-and-drop image uploading
- Three layout views: Grid, Masonry, and List
- Full-screen lightbox with keyboard navigation
- Thumbnail strip for quick browsing
- Delete photos individually

## How to use
Just open `gallery.html` in your browser — no build tools or dependencies needed.

## Tech Stack
HTML · CSS · JavaScript
